# Game of Life

John Conways Game of Life
```
"Alive" cells - "#"
"Dead" cells - " "
```
### Alive cells
- If an alive cell has 4 or more alive cells around it, it dies
- If an alive cell has 1 or less alive cells around it, it dies
- Otherwise, it lives

### Dead cells
- If a dead cell has 3 or more alive cells around it, it becomes alive
